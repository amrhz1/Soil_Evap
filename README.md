All data used in this study are provided in the repository. The main experimental dataset is available in Experimental_Data.xlsx, while the time series data used for Figure 3b can be found in Evaporation_vs_time.xlsx. In addition, sieve analysis results for both fine and coarse soils are included.

The simulation code is provided in column_evaporation_solver_with_sand_mulch_hourly.nb.

The code used for image analysis is available at: https://github.com/jiaqizheng2000/Image_analysis
